# ez-paste

A Mac menu bar app that makes pasting partial screenshots into [Claude Code](https://claude.com/product/claude-code) easy.

## How to Use

1. Click Download
2. Click to unzip
3. Open terminal, and run `xattr -cr ~/Downloads/ez-paste.app`
4. Click on the app file in your Downloads
5. Use `⌃⇧⌘4` to take a screenshot
6. Paste your screenshots directly into Claude Code!

Designed for Apple Silicon.
